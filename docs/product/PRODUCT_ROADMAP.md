# PRODUCT_ROADMAP

## Overview
StageFlow의 단계별 출시 계획을 정의한다.

## Purpose
기술·사업 의사결정을 위한 출시 단계 기준을 제공한다.

## Scope
MVP ~ v2.0

## Goals
- 단계별 scope 명확화
- 의사결정 기준 정립

## Non-goals
- 단기 마케팅 계획
- 세부 일정

## Users
개발 팀, 운영 팀, 외부 연동 파트너

## Requirements
- MVP: 기본 CRUD, API auth, Web, SQLite, Plugin registry
- Alpha: AI 파이프라인, content sync 완성
- Beta: Timeline UI, 실제 Resolume 연동, OpenAPI
- RC: 성능/보안 감사, 다중 환경 배포 검증
- v1.0: 정식 출시, 실제 공연 환경 검증
- v2.0: 멀티 사용자/권한, 클라우드 확장

## Success Metrics
- 단계별 성공 기준 존재
- 각 단계에서 배포 가능 상태 유지
