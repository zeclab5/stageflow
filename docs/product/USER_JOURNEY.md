# USER_JOURNEY

## Overview
사용자가 StageFlow를 통해 이동하는 핵심 작업 흐름을 정의한다.

## Purpose
전체 사용자 흐름을 하나의 흐름으로 이해할 수 있게 정리한다.

## Scope
공연 제작 준비·실행·보관 전 과정

## Goals
- 흐름 단계 명확화
- 각 단계의 입출력 정의
- 결합 지점 식별

## Non-goals
- 세부 화면/프로토타입 정의
- 별도 역할별 예외 흐름

## Users
공연 연출가, 미디어 아티스트, 기술 감독, 공연 기획자

## Requirements
- 프로젝트 생성 → 장면 구성 → 실행 → 보관 흐름
- 필요시 AI/플러그인 단계 포함

## Success Metrics
- 전체 흐름이 8단계 이하로 정리
- 각 단계가 독립 실행 가능
