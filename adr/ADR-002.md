# ADR-002 : Plugin First

Status: Accepted
Date: 2026-07-15

---

# 결정

모든 외부 프로그램 연결은 Plugin으로만 처리한다. 직접 연동 코드를 작성하지 않는다.

---

# 배경

공연 제작에 사용되는 프로그램은 다양하고 버전 변경이 잦다. 직접 연동하면 변경 비용이 크다.

---

# 결정 이유

- 외부 시스템 변경 영향을 격리.
- 기능 확장성 확보.
- 의존성 축소.

---

# 영향

- Plugin Interface는 변경 시 영향 범위가 넓으므로 신중히 설계.
- 초기 설계 비용이 발생하지만 장기 유지보수 비용 절감.
